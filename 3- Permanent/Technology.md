## Apps

- **Next.js:** storefront, tenant dashboard, admin dashboard
- **React Native:** for the mobile app
- **Tailwind**
- **Trpc**: for the backend
- **Postgres via Neon:** for data storage



## Services

- **Turborepo:** to tie everything up as a monorepo.
- **Fly.io:** to host the backend.
- **Vercel:** to host the frontend.
- **Cloudinary:** for image storage.
- **Cloudflare R2:** for file upload (if needed)
- **Cloudflare:** for DNS, CDN, and rate limiting
- **Github Actions + Vercel:** for CI/CD
- **Resend:** for emails
- **Meilisearch:** for search
- **Social media APIs (Facebook, Instagram, TikTok):** for social integrations
